# Image Gallery

## Live Link
(https://images-gallerys.herokuapp.com/)

## Project Description

A simple image gallery made with ReactJS.

### Getting Started

1. Install dependencies `bundle install` from root folder and `npm install` from client folder.
2. Run `npm start` to start app



#### Technologies:
* ReactJS
